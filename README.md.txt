Генератор dad jokes на русском

Вы учите бота шутить, кликая на смешные шутки!

Как работает?
Пользователь вводит начало шутки
LLM1 генерирует три варианта продолжения шутки (панчлайны)
Выбранный вариант записывается в файл .jsonl как образец для LLM2
Так создается few-shot для LLM2 для улучшения качества шуток
Если все три шутки не нравятся, пользователь может кликнуть на кпонку "еще варианты" и получает по три новых панчлайна до тех пор, пока не будет доволен результатом

Как протестировать?
1. Создайте виртуальную среду (рекомендуется)
2. Установите в нее зависимости из requirements.txt
pip install -r requirements.txt
3. Получите ваш groq api-key
4. Создайте файл .env в папке joke_generator c таким содержимым:
GROQ_API_KEY=твой_ключ_здесь
SECRET_KEY=твой_ключ_здесь
ЛИБО
вставьте свой ключ прямо в код:
app.secret_key = "твой_ключ_здесь"
client = Groq(api_key="твой_ключ_здесь")
5. Запустите в терминале python app.py и перейдите по появившемуся адресу на страницу для тестирования
6. Вы великолепны
